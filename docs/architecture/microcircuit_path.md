# Microcircuit-Ausbaupfad

- **Phase A (funktional):** Fokus auf deterministische Pfade und klare Snapshots; Quantisierung und RC-Sortierung sind Pflicht.
- **Phase B (stabilisierend):** Einführung der Microcircuit-Strukturen mit strengen Guardrails für Quantisierung und PVGS-Snapshotting.
- **Phase C (vollständig):** Integration der Microcircuit-Module in alle Fenster- und Overlay-Pfade; tighten-only bleibt unverändert aktiv.

Über alle Phasen hinweg gelten Guardrails für Quantisierung, deterministische Merge-Reihenfolgen und stabile Snapshot-Erzeugung.
